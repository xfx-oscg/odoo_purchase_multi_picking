Purchase Multi Picking

How to use

  • Create PO, choose correct operation type for lines that you what to apply precise operation types. Then confirm PO.


  • PO then will go into "Purchase Order" state.
  • Pickings are generated according to the operation type of PO lines. if there is no operation type for a line, the line will use the picking type of the PO instead, which is the original odoo function.
